# Repo Owner
## Steps
1. Download these files as a zip
2. Create your own repo
3. Create Branch Protections for this repo
4. Add the following rules:
- Create the rule main
- Target the main branch only
- Restrict Deletions
- Require a pull request before merging
- Block force pushes
5. Add collaorators


# Collaborators

## Goal 
Add your name to the contributors list using:

- Branches
- Pull Requests
- Code Reviews

## Steps

1. Clone the repository
2. Create a new branch
3. Add your name to index.html
4. Commit changes
5. Push branch
6. Open Pull Request
7. Review classmates' PRs
8. Merge after approval
